# Customer Segmentation using K-Means

## Overview
This project applies machine learning clustering techniques to segment customers based on income and spending behavior.

## Objective
Identify distinct customer groups to support targeted marketing strategies.

## Methods
- Data visualization
- Feature scaling
- K-Means clustering
- Elbow method for cluster selection

## Key Insights
Five customer segments were identified including premium customers, low engagement customers, and average shoppers.

## Tools Used
Python, Pandas, Seaborn, Scikit-learn
